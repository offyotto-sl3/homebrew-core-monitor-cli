# homebrew-core-monitor-cli

Homebrew tap for [Core-Monitor-CLI](https://github.com/offyotto-sl3/Core-Monitor-CLI).

## Install

```bash
brew tap offyotto-sl3/core-monitor-cli
brew install core-monitor-cli
```

Or in one command:

```bash
brew install offyotto-sl3/core-monitor-cli/core-monitor-cli
```
